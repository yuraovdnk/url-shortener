## Setting up env vars with .env.example 

The `.env.example` file contains examples of environment variables that your application may use. Before using the application, create a `.env` file based on this example and set the environment variable values according to your configuration.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
